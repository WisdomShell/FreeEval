About FreeEval Visualizer

FreeEval Visualizer is a tool designed to help you evaluate and visualize the performance of large language models. It provides a user-friendly interface for comparing model outputs, scoring individual outputs, and analyzing results based on various metrics.

Features

  • Pairwise Comparison: Compare outputs from different models and determine which one performs better.
  • Direct Scoring: Score individual outputs based on predefined criteria.
  • Matching: Evaluate outputs based on their similarity to reference outputs using metrics like accuracy, BLEU, and BERTScore.
  • Comprehensive Analysis: Visualize the distribution of evaluation results and analyze correlations between various metrics.
  • Case Studies: Examine individual cases in detail to understand model performance on specific inputs.
